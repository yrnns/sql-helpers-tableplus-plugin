# Convert Double Quotes Plugin for TablePlus

This is a TablePlus plugin helps you get single quote instead of double quote

# How to Install

1) npm install
2) npm run build
3) open .

# How to Use

1) Right click a table
2) Click "Join" or "Join with Quotes"
3) All done.
